### @ligth ui
component ui for react js made with tailwindcss

### Installation

```bash
git clone https://github.com/chrscalv/lightui.git

```
```bash
npm install
```
```bash
npm run dev
```

### ligth ui directory structure
    |-- .assets
        |--styles
           |--index.css
    |-- components
    |-- pages
        |--_app.js
        |--index.js
    |-- next.config.js
    |--package-lock.json
    |--package.json
    |--postcss.config.js
    |--tailwind.config.js
    |-- README.md
